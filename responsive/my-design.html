My Design

My Design and Build process

My design from the unset was going to be clean and simple, with a very limited colour plate and simple use of typography to guide you through the pages. I first started to plan my design by sketching up layouts of wireframes and placing the elements on the page using a pen and layout pad image 1. Once I was happy with the layout on paper I moved onto building out the layout using HTML 5 and CSS in Sublime Text 2 image 2. Image 3 shows my progression from design to build and image 4 shows a screen shot of the finished design.

The Header

Holds the logo to the left and a tagline to the right of the logo. The main site navigation sits below the logo and tagline in a slightly darker grey just so that it stands out more and to make sure people know that’s the main navigation. Once the user is on a page the dark grey colour is used to indicate that this is the page that you are on, it acts like an active link.

The Login Page

The page opens with a welcome note at the top of the login box. The user is asked to login with a nice “Please Login” pointer with a number 1 in a green circle given the user a helpful tip along the way. The login box uses a darker shade of grey to left it away from the background colour. The login itself is again laid out simple with the use of labels to tell the user what information to plug in to processed, the login button is nice and big and fills must of the space in the fieldset, and works well given the user a big push in the right direction.

My Profile Page

This page uses a 2-column layout, on the left side of the page there is useful information about myself, like my email address and phone etc, also my username and last logged time. On the left hand side is some information about myself with an image of my favorite website.

My Design Page

This page uses a wider container, as it is only a 1-column layout it shows information about the design and the step about how I designed and build the site.

Font Family

I have used Nunito a lovely soft rounded sans-serif typeface I have used this font for all my headings and body text. I have linked this in the head from Google fonts. I have used a nice line-height: 0.8em; this works well even when it scales down.


  • Colour: background-color: #F4F7F8;
  • Main Font color: #384047;
  • H1 Font colour: #384047;
  • H2 Font colour: #4bc970;
  • Nav: background-color: #E3E7E8;
  • Links: #4bc970;
  • Link hover: #333333;

The Footer

A simple border top is used to divide the space with a simple © name and DSA course name.

Responsive Site

I have used a media quire once the browser window goes to 340px it places the profile box on top of the my profile information. I have also made the images scalable I have set them up to be 80% of the space they contain once the window is at 340px.
